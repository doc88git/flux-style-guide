## Flux Style Guide

### Documentação

A documentação, neste momento, se encotra no próprio repositório com o  uso do StoryBook.
```
yarn storybook
```
### Scripts para executar em seu projeto

``` bash
# Instalação
$ yarn add @doc88/flux-style-guide

# Atualização
$ yarn upgrade-interactive --latest
```
